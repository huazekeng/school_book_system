# 校园共享书城

#### 介绍
书城，前后端分离，后端springboot+mybatisplus，前端IView
#### 软件架构
软件架构说明

1.  SpringBoot
2.  MyBatisPlus
3.  Redis
4.  SpringSecurity
5.  Swagger
6.  JWT
7.  IView
